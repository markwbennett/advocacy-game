# Advocacy Game

A trial advocacy game to teach Texas criminal evidence and procedure fundamentals.

## Overview

Players take on the role of a defense attorney representing clients in criminal cases. Through scenario-based gameplay with time pressure, players learn to make real-time decisions on evidence objections, procedural motions, and trial strategy.

## Features

- **Scenario-based learning**: Work through realistic trial situations
- **Time pressure**: Make decisions under the clock, like in real court
- **Progressive difficulty**: Start with Class C misdemeanors, advance to more serious offenses
- **Immediate feedback**: Learn why your decisions were right or wrong with explanations

## Development

### Setup
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```
